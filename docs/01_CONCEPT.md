# 孫っち - コンセプト

> 最終更新: 2024年12月

## プロジェクト概要

**孫っち**は、高齢者向けの音声AIアシスタントWebアプリケーションです。

「話しかけるだけで何でもできる」体験を提供し、デジタルデバイドを解消します。

## 解決したい課題

### 社会的背景
- 日本の65歳以上人口: 約3,600万人（総人口の29%）
- 一人暮らし高齢者: 約670万人（増加傾向）
- スマホ保有率は上昇しているが、使いこなせていない層が多い

### ユーザーの課題
| 課題 | 具体例 |
|------|--------|
| デジタルデバイド | アプリの操作が複雑で使えない |
| 孤独感 | 話し相手がいない、会話の機会が減少 |
| 記憶の補助 | 予定や買い物リストを覚えておくのが難しい |
| 情報アクセス | 調べたいことがあっても方法がわからない |

## ターゲットユーザー

### プライマリターゲット
**65歳以上の高齢者（本人利用）**
- スマホ・タブレットは持っているが、複雑な操作は苦手
- 一人暮らし、または日中一人になることが多い
- 話し相手が欲しい、日常のサポートが欲しい

### ペルソナ

#### 田中 花子さん（78歳・女性）
- 東京郊外で一人暮らし
- タブレットはあるが、LINEと写真閲覧のみ
- 買い物リストや予定をよく忘れる
- 話を聞いてくれる相手がいない

#### 佐藤 一郎さん（72歳・男性）
- 妻と二人暮らし、日中は妻がパートで不在
- ニュースや天気を知りたいが、スマホ操作が億劫
- 通院の予定を忘れがち

## コアバリュー・差別化

### 「話すだけで、なんでもできる」

| 従来のアプリ | 孫っち |
|-------------|--------|
| 画面を見て、ボタンを探してタップ | 「買い物リストに牛乳追加して」 |
| カレンダーを開いて日付選択して入力 | 「来週火曜に眼科の予約入れて」 |
| ブラウザで検索して読む | 「明日の天気教えて」 |

### 競合との違い

| サービス | 孫っちとの違い |
|---------|---------------|
| Alexa/Google Home | 定型コマンド中心、長期記憶なし |
| Cotomo | 雑談特化、タスク管理機能なし |
| ElliQ | 専用ハードウェア必要、日本未展開 |

**孫っちの強み:**
1. 自然言語での操作（定型コマンド不要）
2. 長期記憶（ユーザーのことを覚える）
3. 多機能統合（タスク管理+雑談+情報検索）
4. 高齢者特化UI
5. 日本語最適化
