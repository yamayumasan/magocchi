# 孫っち - MVP仕様

> 最終更新: 2024年12月

## MVPの定義

**MVP = 最小限の価値を提供できる製品**

「高齢者が音声だけで日常のタスクをこなせる」ことを証明する最小セット。

---

## 機能一覧

### 1. 音声会話基盤（必須）

#### 1.1 音声入力
- ブラウザのマイクで音声を録音
- Whisper APIで文字起こし
- 日本語対応

#### 1.2 音声出力
- LLMの応答を音声で読み上げ
- ElevenLabs または VOICEVOX を使用
- 自然な日本語音声

#### 1.3 会話処理
- Claude/GPT APIでの応答生成
- 「孫っち」のペルソナでの応答
- ストリーミング表示（文字が順次表示）

### 2. タスク管理

#### 2.1 買い物リスト
```
ユーザー: 「買い物リストに牛乳を追加して」
孫っち:   「牛乳を追加しました。今のリストは、牛乳、卵、パンの3つです」

ユーザー: 「買い物リストを教えて」
孫っち:   「今の買い物リストは、牛乳、卵、パンの3つですよ」

ユーザー: 「牛乳は買ったから消して」
孫っち:   「牛乳を消しました」
```

#### 2.2 リマインダー
```
ユーザー: 「明日の朝9時に薬を飲むのを思い出させて」
孫っち:   「明日の朝9時に薬を飲むリマインダーを設定しました」

※ MVP段階ではブラウザ通知で対応
```

#### 2.3 予定管理
```
ユーザー: 「来週の火曜に眼科の予約があるから覚えておいて」
孫っち:   「来週の火曜日、眼科の予約ですね。覚えておきます」

ユーザー: 「今週の予定を教えて」
孫っち:   「今週は火曜日に眼科の予約がありますよ」
```

### 3. 情報提供

#### 3.1 天気
```
ユーザー: 「今日の天気は？」
孫っち:   「今日の東京は晴れ、最高気温は15度です。
           少し肌寒いので、上着があるといいですよ」
```

#### 3.2 日時
```
ユーザー: 「今日は何日？」
孫っち:   「今日は12月15日、日曜日です」

ユーザー: 「お正月まであと何日？」
孫っち:   「お正月まであと17日ですよ」
```

### 4. 雑談・話し相手

```
ユーザー: 「最近寂しいんだよね」
孫っち:   「そうなんですね。お話を聞かせてください。
           何かあったんですか？」
```

### 5. 基本UI

#### 5.1 メイン画面
- 大きな「話す」ボタン（画面中央）
- 会話履歴の表示エリア
- シンプルなヘッダー

#### 5.2 設定画面
- 音量調整
- 地域設定（天気用）
- ユーザー名設定

## 非機能要件

### パフォーマンス
- 音声入力から応答開始まで: 3秒以内
- ページ読み込み: 2秒以内

### アクセシビリティ
- 文字サイズ: 18px以上
- ボタンサイズ: 48px以上
- コントラスト比: WCAG AA準拠

### 対応環境
- Chrome（最新版）
- Safari（最新版）
- タブレット対応

---

## MVPに含めないもの

以下は将来フェーズで実装:
- ユーザー認証・ログイン
- 複数の声の選択
- 家族向け見守り機能
- オフライン対応
- ネイティブアプリ

---

## 成功基準

### 定量
- 音声認識精度: 90%以上（標準的な発話）
- 応答時間: 3秒以内
- 1セッションで3タスク以上完了可能

### 定性
- 高齢者が説明なしで「話す」ボタンを見つけられる
- 基本的な操作を5分以内に理解できる
- 「また使いたい」と感じる
