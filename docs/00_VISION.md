# 孫っち - ビジョン・将来像

> 最終更新: 2024年12月

## ビジョン

**すべての高齢者が、テクノロジーの恩恵を自然に受けられる社会を実現する**

## ミッション

音声という最も自然なインターフェースで、高齢者の日常生活をサポートする

## 本質的な価値

### 「孫っち」が提供するもの

1. **デジタルの民主化**
   - 技術的なスキルに関係なく、誰でもテクノロジーを使える
   - 「できない」を「話すだけでできる」に変える

2. **孤独感の軽減**
   - 話し相手としての存在
   - 「聞いてくれる」という安心感

3. **自立の支援**
   - 記憶の補助（買い物リスト、予定、大切なこと）
   - 家族に頼らなくても日常をこなせる自信

4. **家族とのつながり**
   - 高齢者の状況を家族が把握できる（将来機能）
   - 離れていても見守れる安心

## 将来像（3〜5年後）

### Phase 1: 個人向けサービス（現在〜1年）
- Webアプリとして個人利用
- 基本的な会話・タスク管理機能

### Phase 2: 家族向け拡張（1〜2年）
- 見守りダッシュボード
- 家族への通知機能
- 複数人での利用

### Phase 3: B2B展開（2〜3年）
- 介護施設向けパッケージ
- 自治体向け高齢者支援ツール
- カスタマイズ可能なホワイトラベル提供

### Phase 4: プラットフォーム化（3〜5年）
- サードパーティ連携（服薬管理、健康管理など）
- 地域サービスとの連携（配食、見守り訪問など）
- 高齢者向けサービスのハブとして機能

## 目指す世界観

```
「おはよう、孫っち」
「おはようございます。今日は少し肌寒いですね。
 今日の予定は10時に眼科、午後は太郎くんが来る日ですよ。
 買い物リストの牛乳、昨日買いましたか？」
```

高齢者が毎朝、孫に話しかけるように自然に使える。
それが「孫っち」の目指す姿です。
